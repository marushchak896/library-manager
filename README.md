# Library Manager

Веб-застосунок для управління бібліотекою книг та авторів.

## Як запустити

1. Завантажити файл `index.html`
2. Відкрити його у браузері (подвійний клік)
3. Логін: `admin`, Пароль: `admin`

## Функціонал

- Додавання, редагування та видалення книг
- Додавання та видалення авторів
- Пошук книг за назвою або автором
- Експорт списку книг у CSV файл
- Авторизація (логін/логаут)

## Технології

- HTML
- CSS
- JavaScript
- localStorage (збереження даних у браузері)
